# karabiner

这个目录放 Karabiner-Elements 的 complex modifications 规则。

当前提供：

- `global-backspace.json`

用途：

- `Ctrl-h` -> `Backspace`

使用方式：

1. 安装 Karabiner-Elements
2. 把 `global-backspace.json` 复制到 `~/.config/karabiner/assets/complex_modifications/`
3. 打开 Karabiner-Elements
4. 进入 `Complex Modifications`
5. 点击 `Add predefined rule`
6. 启用 `Global backspace remaps`
